# OTR — Mentoring & Git Culture Change

## Case: Внедрение Git (отказ от zip-архивов)

- Problem (S/T):
  - Код передавался по почте zip-архивами
  - Даже тестовые задания присылали по почте с zip
  - Нет code review, нет истории изменений
  - 2 локальных разработчика работали изолированно на своих проектах

- Action (A):
  - Объяснил команде преимущества Git
  - Перевёл 2 разработчиков на GitHub
  - Внедрил cross-code-review между разработчиками
  - Инициировал обмен кодом и обсуждение между командой

- Result (XYZ):
  - Команда из 3 человек перешла на Git
  - Появился code review и обмен знаниями
  - Разработчики получили доступ к чужому коду
  - Культура разработки сменилась с изолированной на совместную

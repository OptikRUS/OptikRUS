# ARTW — Mortgage Calculator Service: Details

## Business Context (уточнённый)

- ТЗ: "Сделать как у Самолёта" (российский застройщик с хорошим ипотечным калькулятором)
- UX: пользователь выбирает объект недвижимости → видит ипотечные предложения банков,
  процентные ставки, первоначальный взнос
- Бизнес-метрики (конверсия, продажи) — неизвестны, не предоставлялись разработчику

## Technical Details (уточнённый)

- Стек: FastAPI + Django Admin
- Feature toggle через админку
- Переписал API client + тесты
- Time-to-market: ~3–4 недели
- Изолированный сервис с доменной логикой

## Key Insight
> Бизнес-эффект: удобство для покупателей недвижимости —
> интеграция банковских условий прямо на карточке объекта
